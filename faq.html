FAQ

Frequent problems and questions.

Q
What USB flash drive should I get?
A

Most USB Flash Drives will work for this tutorial. However, most operating systems today average around 3 Gigabytes in size, so a 4 GB or more flash drive is necessary. Another specification that is good for this tutorial is to get a USB 3.0 flash drive. This is one of the latest versions of USB, and it runs and copies data much more quickly than older versions. If you do not have a flash drive, you can visit Amazon. You can also see if a friend has a spare to use.

Q
What is an ISO file?
A

An ISO file is an archive file of an optical disc. An archive file is just a file that contains one or more (usually many) files inside of it. The most common example of an optical disc is a CD-ROM. The ISO files that this tutorial uses for creating a bootable USB is a file that contains the necessary files to boot up a computer with a certain operating system. For example, a Windows ISO file has the necessary data to boot up a computer to Windows.

For more information, visit this Wikipedia page.

Q
How do I get an ISO file for my operating system?
A

The process all depends on which operating system you are using.

  • Windows

    For a Windows computer, go to this Microsoft Page. Microsoft currently releases ISO files only for Windows 7, 8, and 10. Click on the version of Windows that you are using, and follow the instructions to download the ISO file.

  • Mac OS X

    On a Mac computer, you must go to the app store. The app store has the various version of Mac OS X available in ISO form.
    Note: You may not get an ISO file from Windows or Linux. You must have a Mac.

  • Linux

    In order to download Ubuntu, go to this link. This page is specifically for Ubuntu, a version of Linux. For another specific version (Fedora or Mint), a simple Google search will pull up the necessary pages to download an ISO file. Follow the instructions on the page, and it will lead you to a download.

Q
What is the Boot Menu?
A

The Boot Menu is a simple menu that shows up when a computer is started under certain conditions. Most of the time, a PC requires the user to repeatedly press a function key (F1) to enter this menu. When the menu opens, the user can select where to boot from, whether it be a network drive, USB device, or hard drive.

For more information, visit this page.
