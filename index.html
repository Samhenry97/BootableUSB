Create A Bootable USB Drive.

A simple guide to creating and using a bootable USB drive.

Get Started

Why?

Imagine you're sitting at your computer, and suddenly it shuts off without notice. You press the power button, and it starts up, but shows an error message. Your operating system has crashed! This is one of the main reasons to create a bootable USB drive. If the operating system crashes or your computer becomes unusable, then a bootable USB drive has the power to reinstall the entire operating system. Once the operating system is reinstalled, the computer will be up and running again! A second reason to create a bootable USB is to speed up your computer. It will reinstall the operating system, causing your computer to run faster because it has been freshly installed. Overall, this skill is a good one to have for many sticky situations.

What Do I Need?

 
  • A USB Flash Drive, preferably 4 GB or more
  • A working computer with a USB port
  • An ISO file of your operating system
  • A few hours of your time
  • A strong internet connection

How?

Normally, creating a bootable USB drive involves a separate program that handles the hard things. The process is different on every operating system. Windows has a built-in tool for developers, but better alternatives exist to make the process easier. Mac users also have a built-in command-line utility. But again, a program exists to make the process easier on Mac. Linux users use a program to make the process simple and quick. To get started, select your operating system from the list below. The only operating systems supported are Windows, Mac OS X, and Linux.
