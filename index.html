Create A Bootable USB Drive.

A simple guide to creating and using a bootable USB drive.

Get Started

Why?

There are many reasons to create a bootable USB drive. One of the major reasons is to fix a crashed computer. For example, if your computer crashes and becomes unusable, then a bootable USB drive has the power to reinstall the entire operating system. Once the operating system is reinstalled, the computer will be up and running again! Another reason to create a bootable USB is to speed up your computer. It will reinstall the operating system, so you will have to back up your files before trying this. Overall, this skill is a good one to have for many sticky situations.

How?

Creating a bootable USB drive is different on every operating system. Windows has a built-in tool for developers, but better alternatives exist to make the process easier. Mac users must use a command-line utility, making things slightly harder to do. Linux users must also use a few command-line options to create a bootable USB. To get started, select your operating system from the list below. The only operating systems supported are Windows, Mac OS X, and Linux. This site should detect what operating system you are using. You do not have to choose your operating system, though.

What Do I Need?

  • A USB Flash Drive, preferably 4 GB or more
  • A working computer with a USB port
  • A few hours of your time
  • A strong internet connection
