Linux Guide

A simple guide for creating a bootable USB drive on Linux.

1

Open the Ubuntu Software Center, as shown in the image below.

The software center should look similar to the image below when it opens.

 
2

In the search bar on top, type in "unetbootin" and press Enter

Once the result shows up, press Install on the right side.

A window will pop up asking for your password. Type it in and press Okay to authenticate. This will install UNetbootin to your computer, which is the tool that we will use to create the bootable USB device.

 
3

Once UNetbootin has finished installing, an icon should appear in your dock on the left.

Click on this icon to open UNetbootin. It should open a window like the one below.

 
4

Select the Diskimage button on the window.

 
5

Click on the ... button to open the file chooser.

When the file chooser comes up, navigate to the ISO file that you downloaded for your operating system. Select it, and then press Open.

 
6

Click on the device dropdown and select your USB device if it is not already selected.

 
7

Click on Okay to start the process.

Caution: Be sure all critical data is backed up on the device, as this process will wipe the flash drive.

 
8

UNetbootin will now create your bootable USB device. It will take a few minutes for this process to complete. It may appear to freeze in the middle of the process. If this freezing happens, just wait patiently for it to finish.

 
9

When UNetbootin finished, select Reboot to restart your computer right away, or press Exit if you want to wait before restarting.

 
10

The last step is to boot your computer using the flash drive you just created. This process will vary from computer to computer, and will not be covered in this tutorial. Most PC's will have you repeatedly press a function key (F1) when the computer is booting up to enter a Boot Menu. Here is a list of helpful links to get started with this:
