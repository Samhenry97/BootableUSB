1

Open the Ubuntu Software Center, as shown in the image below.

The software center should look similar to the image below.

2

In the search bar on top, type in "unetbootin" and press Enter

Once the result shows up, press Install on the right side.

A window will pop up asking for your password. Type it in and press Okay to authenticate. This will install UNetbootin to your computer, which is the tool that we will use to create the bootable USB device.

3

Once UNetbootin has finished installing, an icon should appear in your dock.

Click on this icon to open UNetbootin. It should open a window like the one below.

4

Select the diskimage button on the window.

5

Click on the ... button to open the file chooser

When the file chooser comes up, navigate to the ISO file that you downloaded for your operating system. Select it, and then press Open.

6

Click on the device dropdown and select your USB device (if it is not already selected).

7

Click on Okay.

8

UNetbootin will now create your bootable USB device. It will take a few minutes for this process to complete.

9

When UNetbootin finished, select Reboot to restart your computer right away, or press Exit if you want to wait before restarting.
