Mac OS X Guide

A simple guide for creating a bootable USB drive on Mac OS X.

1

Download and open UNetbootin from this link.

UNetbootin is a free program for Mac and Linux that allows the quick creation of a bootable USB device.

2

Open the downloaded file from Safari. The file should look similar to the image below.

Double-Click on UNetbootin to open it.

You may run into an error here. If the error box below shows up, then press the Control key on your keyboard and right-click UNetbootin at the same time. When the dropdown menu shows up, click Open.

Now another box should show up asking for permission to run UNetbootin. Click Open to confirm.

Enter your administrator credentials to authenticate.

3

When UNetbootin opens, select the Diskimage button, shown in the picture below.

4

Select the ... button to open the file chooser.

Navigate to the ISO file that you downloaded and select it.

5

Select your USB device from the dropdown shown in the image below if it is not already selected.

Note: If you are not sure which device is your flash drive, open a Terminal window and type in diskutil list. This command will list all of the devices connected to the computer.

6

Select Okay to start the process.

Caution: Be sure all critical data is backed up on the device, as this process will wipe the flash drive.

7

UNetbootin will do the rest for you.

Note: This process may take up to ten minutes. It may also appear to freeze briefly in the middle of the process. If this freezing happens, let it sit for some time, and it should finish.

8

When UNetbootin has finished, press Exit to shut it down.

9

The last step is to boot your computer using the flash drive you just created. This process will vary from computer to computer, and will not be covered in this tutorial. A Mac cannot be dual booted with Linux, so you must boot with a PC. Most PC's will have you repeatedly press a function key (F1) when the computer is booting up to enter a Boot Menu.
